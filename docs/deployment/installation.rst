Installation
============

1. Clone the repository
-----------------------

.. code-block:: bash

   git clone https://github.com/Systems-Immunometabolism-Lab/GUANACO_updated
   cd GUANACO_updated

2. Install from local directory
-------------------------------

.. code-block:: bash

   pip install .

For development (editable install):

.. code-block:: bash

   pip install -e .
